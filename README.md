# AirGuardian-AI-Project-# 🌿 AirGuardian AI

AirGuardian AI is a multi-agent AI-powered Air Quality Management System that monitors air pollution, predicts future AQI, and provides personalized health recommendations. The platform helps users make safer daily decisions while promoting environmentally responsible actions.

🚀 Features

* 🌍 Real-time AQI Monitoring
* ❤️ Personalized Health Advice
* 🔮 AQI Forecasting & Alerts
* 🌱 Community Action Suggestions
* 🤖 AI Assistant with Quick Prompts
* 📱 Responsive Web Interface

🛠️ Tech Stack

* HTML5
* CSS
* JavaScript
* IBM watsonx.ai
* Granite Models 
* LangFlow 

🏗️ AI Agents

*AQI Data Agent** – Collects and analyzes air quality data.
  *Health Advisory Agent** – Provides personalized health recommendations.
  *Forecasting Agent** – Predicts future AQI and pollution risks.
  *Community Action Agent** – Suggests eco-friendly actions and awareness initiatives.
